修改 Todo 是否完成

Url/api/Todo/UpdateTodoStatus
HttpStatusCodePut

Request

Body

欄位名稱(英文)欄位名稱(中文)型態列舉驗證
TodoIdPKguidf79418cb-bb8e-4425-8028-383949303834必輸
IsComplete是否完成string必輸 , 只能Y與N
{
  "todoId": "f79418cb-bb8e-4425-8028-383949303834",
  "isComplete": "Y"
}

Service

  1. 是否有此筆資料 ⇒ ReturnCode 4001
  1. 當TodoStatus = Y，請將CompleteTime 預設為現在時間
  1. 當TodoStatus = N，請將CompleteTime 預設為null

Response

{
  "returnCode": 2000,
  "returnMessage": "修改成功",
  "returnData": null
}
